# Mac OS X Setup

##System Settings

###General
* Use Dark Theme
* Recent items: none

###Dock
* Automatically hide and show: on
* Show recent apps: off

###Security & Privacy
* Require Password immediately
* Turn on FireVault

### Notification
* Turn on Do Not Disturb

###Network
* Change DNS to [1.1.1.1](https://1.1.1.1/dns)

###Keyboard
* Key Repeat: Fastest
* Delay Until Repeat: Fastest

###TrackPad
* Tap to Click: on

###Safari
* Enable Developer tools

##Terminal

* Install [HomeBrew](https://brew.sh)
* Git Clone Dotfiles
  * .zshrc
  * .vimrc
* Install Brew Leaves
* Install Brew Casks
